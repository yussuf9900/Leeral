# Mobbin Design System Guidelines

This skill implements elite visual layouts and spacing inspired by the best real-world mobile and web applications cataloged in Mobbin.

## Rules
- **Grid Layouts**: Construct balanced, highly structured layouts using standard flex-rows or css-grids.
- **Card Styling**: Rounded corners (`border-radius: 16px`), thin borders, and structured paddings (`padding: 24px 30px`).
- **Interactive Widgets**: Replaced static pictures/drawings with functional dashboard widgets, complete with dynamic numbers and mini-charts.
- **Modals**: Standardize custom overlay modals with dark backdrops to handle detail-view sheets rather than utilizing generic browser alerts.
- **Consistency**: Keep layout menus, header structures, icons, and buttons unified across all routes.
